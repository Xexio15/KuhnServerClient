# A2
### Parametros de ejecucion del Servidor  
-p \<puerto\> -i \<modo 1|2\>  
  1: IA aleatoria  (Predeterminado)
  2: IA optima   
    
### Parametros de ejecucion del Cliente
-s \<IP\> -p \<puerto\> -i \<modo 0|1|2\>  
  IP: localhost si es en un mismo equipo  
  0: Jugador real  (Predeterminado)
  1: IA aleatoria  
  2: IA optima
